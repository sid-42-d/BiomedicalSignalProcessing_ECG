\# Dataset



The dataset used in this project contains multi-channel physiological signals including:



\- ECG

\- PPG

\- Dynamometer



Due to GitHub repository size best practices, the dataset is hosted externally instead of being uploaded directly.



Download the dataset from the link below:



Google Drive Link:

https://drive.google.com/file/d/1PaDHAhDhR5QUOnqH7Oq0KLn4rM2TeZnk/view?usp=sharing



After downloading the dataset, place the file inside the `data/` folder before running the code.
